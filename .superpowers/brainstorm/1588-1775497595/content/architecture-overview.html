Key Design Decisions

Chat Path (Qwen)

  • Streaming responses for snappy UX
  • Tool-calling for scene control
  • Stateless per request (context in messages)
  • High volume, low cost per call

Agent Path (Claude)

  • Deep multi-step reasoning
  • Persistent memory (Vercel KV)
  • Publishes threat pages & briefings
  • Low volume, high value per call
